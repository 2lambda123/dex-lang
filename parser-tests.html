For now, arithmetic is not sensitive to whitespace:

:p 1.0+1.0
2.0
:p 1.0 +1.0
2.0
:p 1.0+ 1.0
2.0
:p 1.0 + 1.0
2.0
:p 1.0-1.0
0.0
:p 1.0 -1.0
0.0
:p 1.0- 1.0
0.0
:p 1.0 - 1.0
0.0

Applying a function to a negative literal thus requires parentheses.

f = \x. x + 10.
:p f -1.0 -- parses as (-) f (-1.0)
Type error: Expected: (Real -> Real) Actual: Real :p f -1.0 -- parses as (-) f (-1.0) ^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
:p f (-1.0)
9.0
